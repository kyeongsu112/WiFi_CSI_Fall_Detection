# WiFall MVP

## 하네스: wifall-mvp

**목표:** WiFall 데이터셋 기반 낙상 감지 모델 학습과 리플레이 웹 대시보드를 end-to-end로 구현한다.

**트리거:** 구현, 개발, 학습, 대시보드, 파이프라인, 코드 작성 등 이 프로젝트의 개발 작업 요청 시 `wifall-orchestrator` 스킬을 사용하라. 단순 코드 질문이나 설명 요청은 직접 응답 가능.
